\block{1. Project Overview}{
    The \textbf{Makeitalive} project explores the transformation of a static landscape
    photograph into a realistic animation.
    We preserve the original texture while injecting coherent, loopable movement.

    \vspace{0.8em}
    We investigated two complementary approaches:
    \begin{itemize}
        \item \textbf{Motion-Flow U-Net}: a lightweight CNN that predicts a dense optical
              flow field $(dx, dy)$ per pixel; pixels are warped over time to create motion.
        \item \textbf{SVD fine-tuning}: a Stable Video Diffusion model fine-tuned on
              landscape drone footage to generate full video frames directly.
    \end{itemize}
}

\block{2. The Motion-Flow Concept}{
    The approach estimates an \textit{Optical Flow} field from a single image:
    \begin{itemize}
        \item A U-Net predicts per-pixel displacement vectors $(dx, dy)$.
        \item \textbf{Backward Mapping} warps the original image at each time-step $t$:
              \[
                I_{\text{anim}}(x,y,t)
                  = I_{\text{orig}}\!\bigl(x - \alpha_t\,dx,\;y - \alpha_t\,dy\bigr)
              \]
        \item $\alpha_t$ is changed to produce seamless loops.
        \item Boundary condition \texttt{mode='reflect'} avoids edge artifacts.
    \end{itemize}

    \begin{center}
        \includegraphics[width=0.85\linewidth]{motion_flow_grid.png}
    \end{center}
}

The U-Net architecture with skip connections is used to maintain spatial resolution.

\block{4. Rendering \& Inference Strategies}{
    Once the flow is predicted, two rendering strategies are used to animate the image:

    \vspace{0.8em}
    \textbf{A. Oscillatory Warping (Seamless Loop)}
    \begin{itemize}
        \item Uses a \textbf{single flow prediction} from the original image.
        \item The displacement is modulated by an oscillating function (e.g., sine wave).
        \item \textbf{Advantage:} Perfectly seamless cycles; preserves 100\% of original texture detail as it always warps from the source frame.
    \end{itemize}

    \vspace{0.8em}
    \textbf{B. Auto-Regressive Inference (Recursive)}
    \begin{itemize}
        \item The model's output becomes its next input in a recursive loop:
        \[ I_{n+1} = \text{Warp}(I_n, \text{Model}(I_n)) \]
        \item \textbf{Advantage:} Generates \textbf{continuous, non-periodic motion} that can evolve over time.
        \item \textbf{Trade-off:} Repeated resampling may lead to accumulated "blurring" after many steps.
    \end{itemize}
}

\block{5. The SVD Architecture}{
    \textbf{Design rationale.}
    The SVD pipeline conditions a Temporal U-Net on a single image via a dual encoder
    (VAE + CLIP). Only the temporal attention layers are fine-tuned; spatial attention
    weights remain frozen.

\block{7. Result Evaluation}{
    \textbf{Qualitative analysis} (Input $\to$ Generated $\to$ Ground Truth):
    \begin{itemize}
        \item \textbf{Motion Flow strengths:} excellent on water surfaces and
              smooth reflections where flow is coherent.
        \item \textbf{Motion Flow limitations:} flow ambiguity on dense clouds;
              some artifacts on high-frequency textures.
        \item \textbf{SVD\_Xtend:} produces coherent drone-style camera motion;
              $\Delta\text{loss} \approx -0.30$ vs.\ naive LoRA.
        \item \textbf{SVD limitations:} interpolation artifacts on foliage;
              motion amplitude constrained by \texttt{motion\_bucket\_id}.
    \end{itemize}

    \vspace{0.5em}
    \textbf{Ablation --- impact of dataset quality:}

\block{8. Conclusion \& Perspectives}{
    The hybridization of both approaches delivers the best of both worlds:
    \textbf{structural speed} (Motion Flow, real-time, lightweight) combined
    with \textbf{generative richness} (SVD, photorealistic motion).

    \vspace{0.5em}
    \textbf{Future work:}
    \begin{itemize}
        \item \textbf{Semantic segmentation}: apply distinct flow fields to
              sky vs.\ water vs.\ vegetation.
        \item \textbf{Multi-scale inference}: improve flow accuracy for
              fine-grained details.
        \item \textbf{Temporal refinement}: auto-regressive prediction for
              smoother, longer loops.
        \item \textbf{SVD + Flow hybrid}: use Motion Flow to seed SVD
              conditioning for guided generation.
    \end{itemize}
}
