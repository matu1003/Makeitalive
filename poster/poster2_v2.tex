\begin{columns}
    \column{0.5}
    \block{1. Extraction Strategy}{
        The pipeline automates the creation of image pairs $(I\_A, I\_B)$:
        \begin{itemize}
            \item \textbf{Source:} 4K YouTube videos (Landscapes).
            \item \textbf{Sampling:} Extraction every 5 seconds.
            \item \textbf{Frame Gap:} 3-frame interval for fluid motion.
        \end{itemize}
    }

    \block{2. Intelligent Data Filtering}{
        Avoid "cuts" using two robust metrics:
        \begin{itemize}
            \item \textbf{Hist Distance}: Bhattacharyya ($>0.35$).
            \item \textbf{Thumbnail MAE}: Structural difference ($>0.25$).
        \end{itemize}
        This filtering ensures the model only learns real movement and not montage jumps.
    }

    \block{3. Architecture Parameters}{
        \begin{itemize}
            \item \textbf{Input:} $512 \times 512$ (Center Crop).
            \item \textbf{Optimizer:} Adam ($10^{-4}$).
            \item \textbf{Loss:} Mean Squared Error (MSE) on flow.
            \item \textbf{Epochs:} 100 on a dedicated local dataset.
        \end{itemize}
    }

    \column{0.5}
    \block{4. Qualitative Analysis}{
        \begin{itemize}
            \item \textbf{Strengths:} Excellence in water and reflections.
            \item \textbf{Limitations:} Some artifacts on dense clouds due to flow ambiguity.
            \item \textbf{Edges:} \texttt{mode='reflect'} essential for aesthetics.
        \end{itemize}
    }

    \block{5. Performance (Loss Convergence)}{
        \begin{center}
        \begin{tabular}{|c|c|c|}
            \hline
            \textbf{Epoch} & \textbf{Training Loss} & \textbf{Validation Loss} \\
            \hline
            1 & 0.452 & 0.480 \\
            10 & 0.120 & 0.145 \\
            50 & 0.045 & 0.052 \\
            100 & 0.038 & 0.041 \\
            \hline
        \end{tabular}
        \end{center}
    }

    \block{6. Perspectives}{
        \begin{itemize}
            \item \textbf{Semantic Segmentation}: Apply specific flows to sky vs. water.
            \item \textbf{Multi-scale Inference}: Improve accuracy for details.
            \item \textbf{Temporal Refinement}: Auto-regressive prediction for smooth loops.
        \end{itemize}
    }

    \block{7. Impact}{
        The project demonstrates that a lightweight architecture can produce qualitative animations with very little training data.
    }
